## 基于中移物联的onenet的智能控制器

### 硬件说明
1、开发板使用ST官网提供的STMB997
2、网络连接使用esp8266wifi模块，使用其AT指令
3、目前实现了通过EDP协议正常接入onenet
### 固件软件说明
1、IAR开发环境
2、UCOSIII操作系统
